# SolRacer Demo Guide

Thank you for checking out SolRacer.

SolRacer is a real-time multiplayer racing game built on Solana using MagicBlock Ephemeral Rollups.

## Gameplay Loop

1. Create or join a race
2. Both players enter the race lobby
3. Players compete in a real-time race
4. Race results are verified on-chain
5. Winner receives the prize

## Demo APK

You can download the APK here:

solracer-demo.apk

Install it on an Android device to try the game locally.

## Controls
LONG graph btn to go forward and roll clockwise in the air.
SHORT graph btn to go backwards and roll anticlockwise in the air.
HOLD is basically to stop both the wheels.

## Competative mode objectives 
• Finish the race faster than your opponent to win

## Practice mode
• The game allows the player to practice on real chart data before hopping into competative mode

## Tech Highlights

• Charts created using real oracle price data
• Solana smart contracts  
• MagicBlock Ephemeral Rollups for real-time multiplayer  
• On-chain race settlement  
• Trustless prize distribution

##Note
- To run the game a seeker device is required, or alternatively Mock MWA wallet needs to be installed on an Android device to run the game. IOS is not yet supported.
